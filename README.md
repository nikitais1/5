# Проект по БД
## Цель проекта:
> * Получить данные о компаниях и вакансиях с сайта hh.ru, спроектировать таблицы в БД PostgreSQL и загрузить полученные данные в созданные таблицы.
## Как работает:
- [ ] В файл database.ini прописываете свои данные для подключения к базе данных.
-

- [ ] Словарь с данными для подключения к БД мы получаем из функции config, находящейся в файле config.py.
- 
- [ ] Полученные словарь передаем в качестве второго параметра в экземпляр класса DBManager (первый параметр - <название базы данных>).
- 
- [ ] У экземпляра класса вызываем метод connect_db(), который создает базу данных и подключается к ней (если таблица с таким именем существует, то этот метод удаляет её).
- 
- [ ] У экземпляра класса вызываем метод create_tables(), который создает таблицы "companies" и "vacancies".
- 
- [ ] У экземпляра класса вызываем метод write_info_in_table(), который заполняет созданные таблицы данными.
- 
- [ ] Данные, упомянутые выше, благодаря функциям get_info(), get_vacancy() и delete_symbol() достаются и записываются в словарь из HeadHunter с помощью API 
- 
- [ ] Далее пользователю предлагают выбрать один из методов, который делает запрос к базе данных и выводит результат на экран.
